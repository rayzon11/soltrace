SolTrace — Solana Wallet Activity Explorer
Trace any Solana wallet instantly. Paste an address and see categorized transactions, SOL flow stats, and shareable links — no wallet connection required.
Features

Fetches last 25 transactions from Solana mainnet RPC
Auto-categorizes: Sent, Received, Swaps, NFTs, Program Calls
SOL in/out summary stats
Filter transactions by type
One-click Twitter share
Links to Solscan for full transaction details
Mobile friendly

Stack

Vanilla HTML, CSS, JavaScript
Solana JSON-RPC API (mainnet-beta)
Zero dependencies, no build step

Getting Started
Open index.html directly in your browser — no install needed.
Or deploy to any static host:
bash# Vercel
vercel deploy

# Netlify
netlify deploy --dir .
Usage

Paste any Solana wallet address into the search bar
Click Trace
Browse the activity feed and filter by transaction type
